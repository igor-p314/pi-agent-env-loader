# Changelog

## [0.1.8] - 2026-04-28

### Добавлено
- `/env list` без пути показывает все установленные переменные окружения
- `/env list <path>` показывает переменные из файла
- `/env get KEY <path>` получает переменную из файла
- Автозаполнение путей стало регистронезависимым

### Изменено
- Синтаксис команд: было `<path> list` → стало `list <path>`
- Синтаксис команд: было `<path> get KEY` → стало `get KEY <path>`
